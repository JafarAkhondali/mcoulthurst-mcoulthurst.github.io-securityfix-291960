Office for National Statistics

National Wellbeing Wheel

As part of the UK's Measuring National Well-being programme, the design team at the Office of National Statistics had developed a novel interactive graphic. This was used to display the current status of around 40 different measures of the Nation's wellbeing. As the project is still in its infancy and due to run for ten years, there are stil some changes to the actual measures being used, as well as the planned monitoring and updating of these measures. The initial wheel had been created in Flash, so I made a number of changes to make the actual content more dynamic, allowing the wheel to be easily customised and more flexible. I also added a dynamic chart element using an external data source. The chart had some basic configuration properties that allowed the team to annotate and update it directly.

One area that I felt needed some improvement was the labelling of the measures. The official wheel uses the correct statistical terminology to label the measures, which I found a little complex given the lighter, more playful nature of the toolkit. My personal preference was to use the labelling from the data source spreadsheet. This improves the readability of the wheel and also makes it more human. Both version are presented here: the earlier May 2013 version and the latest September 2013 version. Note the September verison also features the chart data for some of the measures.

The offical interactive wheel (September 2013) can be found on the ONS website. NB: The data used in the charts below is for testing purposes. The latest wellbeing measure data at the time of writing, can also be found on the ONS website.
