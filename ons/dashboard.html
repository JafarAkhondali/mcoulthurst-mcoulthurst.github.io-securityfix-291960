Office for National Statistics

Prototype Dashboard of Economic Indicators

The Office for National Statistics publishes a massive amount of data every single day. As part of the work on an interactive piece about "How ONS statistics explain the economy" I built a quick prototype to help demonstrate how dynamic data can be used in an application. In this version all the content is dynamically read from a separate data file. The data is processed and each panel is dynamically created in the browser. In the final application, specific time-series data (produced by the ONS) was used to display the charts. This data can then be updated as new data becomes available.
