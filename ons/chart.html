Office for National Statistics

Prototype chart generator

As part of my work at the Office for National Statistics, I spent some time creating prototype toolkits to help improve the creation of assets for the website. The ONS publishes a large number of documents and as part of improving the dissemination of this data, I created a small application to enable members of staff to produce standardised charts for the bulletins. There were two aims: the first was to improve how these charts were created. The secondary aim was to standardise the look of the charts to help establish the ONS branding. This in turn, would help reflect the quality of the work done by the statisticians.

Very simply, the application loads in a CSV data file which is then processed and displayed (using the HighCharts library). The type of chart can by selected to find the most appropriate chart type for the data. Basic titles can then be added to the chart. This final graph can then be exported as an image and embedded in the webpage.
