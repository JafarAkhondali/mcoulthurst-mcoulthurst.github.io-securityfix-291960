The Twitter Accounts

Data Visualisation Census 2013

Again, more proof-of-concept work, building towards a vague Twitter dashboard application. Using the Twitter account names from the Census Data, I built a quick-and-dirty page to access the Twitter API and get a snapshot of the account details (July 2013). Given the 100 account lookup limit, this meant a dozen seperate calls. The JSON data returned is 'fully-hydrated' (This is an example of my account mcoulthurst data) so I then pulled out a few key fields using javascript, before saving this data as a seperate JSON file.

The section below loads in the external data and then creates a card for each account. The image and username link directly to the person's twitter account. There are 1177 account in total and may take some time to load...
