Focus of Intent

Visualisation of “The Census” (March 2013)

I spent some time looking at the "Focus of Intent" question data, trying to find a way to visualise the multiple options in the answers. Here are three versions: A simple order bar chart to show the various combinations; an experimental timeline to try to match up the combinations, by showing the counts for each combination split across the individual values; and the original cumulative counts.

In a seperate experiment, I split the timestamp of the census data into daily counts, in a attempt to see if there was a pattern (there wasn't). I've assumed that the timestamp inthe data is the central server time and so consistent across all entries. If so, then one other idea was to factor in the location to get the local time.

Another aside: The site has now been migrated to using Jekyll, which means I can add a page and the associated navigation elements in record time. The tricky part was figuring out how to create custom javascript tags for different pages.
