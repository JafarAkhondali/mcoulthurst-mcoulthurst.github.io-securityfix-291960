The Survey

Visualisation of “The Census” (March 2013)

Based on Andy Kirk's Census of Data Visualisation practitioners from March 2013, I built a quick set of visualisations. The aim of this first draft was to get the basic charts up and running and see what the data actually looked like. It was also intended as a way of developing my d3.js knowledge.

The only change to the data was to clean up the location field to include a country and to standardise that description to be able to reference open data (see the animated map). The initial draft charts included bar charts for all the data sets, except for a pie chart(!) for the Extent of an interviewee's interest (this was chosen purely as an exercise in creating one in d3.js).

The charts were then refined, with the demographic changed into the more standard population pyramid. The 'Timeline' was altered into a line chart in order make the low numbers more readily apparent. The pie was replaced with an ordered bar chart. Rollover functions were added to the 'Demographic' and 'Experience' charts.

The longer term plan was to add more functionality to allow filtering of the data and the extraction of Twitter details. (To be continued...)
